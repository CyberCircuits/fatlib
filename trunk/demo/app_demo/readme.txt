fatlib App Demo
---------------

This is a simple demo showing:

* Building an app 
* Using macro processes
* Registering and switching screens
* Building simple screens
* Triggering mapped processes


FlashDevelop users
------------------

* Open as3proj file
* Download the fatlib library
* Add a classpath to fatlib
* Compile and run

Other users
-----------

* Make a new project
* Download the fatlib library
* Add classpaths to src and fatlib
* Compile and run Main.as
